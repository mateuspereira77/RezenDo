# Pasta de Imagens para a Página de Ajuda

Esta pasta contém as imagens/screenshots usadas na página de ajuda (`/ajuda`).

## Imagens Necessárias

Adicione as seguintes imagens nesta pasta:

1. **dashboard-principal.png** - Screenshot do dashboard principal
2. **formulario-criacao.png** - Screenshot do formulário de criação de tarefa
3. **modal-edicao.png** - Screenshot do modal de edição
4. **calendario-semanal.png** - Screenshot do calendário semanal com barra de rolagem
5. **compartilhar-tarefa.png** - Screenshot da seção de compartilhamento
6. **comentarios-menciones.png** - Screenshot mostrando comentários e menções
7. **historico-tarefas.png** - Screenshot da página de histórico
8. **produtividade.png** - Screenshot da página de produtividade com gráficos

## Formato Recomendado

- Formato: PNG ou JPG
- Tamanho: Recomendado 1200px de largura (ou maior)
- Qualidade: Boa qualidade para leitura de textos na interface

## Como Adicionar

1. Capture screenshots da aplicação
2. Salve as imagens nesta pasta com os nomes exatos listados acima
3. As imagens aparecerão automaticamente na página de ajuda
